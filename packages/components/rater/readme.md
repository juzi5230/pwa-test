### Rater

> 评分器，由于是字符实现的，不同浏览器可能看起来会有一些不同。

### Props

|name|type|default|description|
|----|----|-------|-----------|
|max|Number|5|最多可选个数|
|value|Number|0|值，使用 v-model 绑定|
|disabled|Boolean|false|是否禁用|
|star|String|★|字符|
|active-color|String|#fc6	|选中时的颜色|
|margin|Number|2|间隙值|
|font-size|	Number|25|字体大小|
